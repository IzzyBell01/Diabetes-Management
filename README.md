# Diabetes Meal Detection Project

This project develops and evaluates machine learning models for detecting meal events from Continuous Glucose Monitoring (CGM) data.

## Folder structure
- **src/**: source code (preprocessing, windowing, models, utilities)
- **data/**: raw and processed datasets
- **results/**: model outputs and figures
- **notebooks/**: for exploratory analysis

## Usage
Run the main pipeline:
```
python src/main_pipeline.py
```
